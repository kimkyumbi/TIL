# 230814-1 / SWIFT / TextField

TextField는 사용자의 입력을 받을 수 있게 해 준다.

```swift
struct Day5: View {
    
    @State var userID: String = ""
    
    var body: some View {
        VStack(alignment: .leading) {
            Text("ID")
            TextField("Enter your ID", text: $userID)
        }
        .padding()
    }
}
```
