# 230818-1 / SWIFT / Toggle

```swift
struct Day7: View {
    
    @State var isLightOn: Bool = false
    
    var body: some View {
        Toggle(isOn: $isLightOn) {
            if isLightOn {
                Text("Light On")
            } else {
                Text("Light Off")
            }
        }
        .toggleStyle(.switch)
        .tint(.green)
        .padding()
    }
}
```

Toggle은 우리가 쉽게 볼 수 있는 스위치를 끄고 키는 것과 같은 동작을 한다.
