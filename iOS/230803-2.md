# 230803-1 / SWIFT / Text
```swift
VStack {
    Text("Hello kyumbi!")
        .bold() // 굵은 글씨
        .italic() // 이탤릭체로 바꿔줌
        .strikethrough() // 글씨 가운데에 줄
    Text("Hello kyumbi!")
        .font(.system(size: 30)) // 글씨 크기 지정
    Text("Hello kyumbi!")
        .underline(true, color: .orange) // 밑줄
        .foregroundColor(.red)
        .background(.purple) // 뒷배경색 지정
    Text("Hello kyumbi!")
        .foregroundColor(.gray) // 글씨 색 지정
        .font(.system(size: 39, weight: .bold, design: .rounded))
            
    }
```
